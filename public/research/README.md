# Research PDFs

Drop submission and report PDFs here. Anything in this folder is served from
the site root under `/research/`.

For example, a file saved as:

    public/research/ai-education-equity-2025.pdf

is linked from a content file's frontmatter as:

    pdf: "/research/ai-education-equity-2025.pdf"

Keeping the PDFs in the repo means the download links are permanent and travel
with the site.
